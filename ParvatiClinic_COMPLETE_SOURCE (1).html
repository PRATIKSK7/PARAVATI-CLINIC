📈 Logistic Regression
Linear classifier using softmax for multiclass. Finds best hyperplane separating 30 disease classes. Top performer due to L2 regularization across 31 binary symptom features. max_iter=2000, C=1.
🌲 Random Forest
Ensemble of 300 decision trees using bootstrap aggregation. Reduces variance via majority voting. Limited by inherent symptom-disease ambiguity in the dataset. n_estimators=300.
⚡ Gradient Boosting
Sequential ensemble where each tree corrects previous residuals. learning_rate=0.1, n_estimators=100, subsample=0.8. Strong on structured tabular data but sensitive to noisy labels.
📊 Naive Bayes
Multinomial NB (alpha=0.5) using Bayes' theorem with feature independence assumption. Fast training, naturally handles symptom frequency vectors well despite independence assumption.
